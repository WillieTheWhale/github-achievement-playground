# GitHub achievement playground

A transparent sandbox for learning GitHub workflows and profile achievements.
All activity here is intentionally isolated from production repositories.

## Experiments

- Issue #1 tested issue creation, a heart reaction, and a prompt closure.
- Discussions #2 and #3 tested Q&A answers and acceptance behavior.
- Discussions #6 and #7 used GitHub Actions as the asker and answer acceptor;
  their two accepted human answers earned Galaxy Brain.
- Pull requests #4 and #5 recorded both directions of a human/GitHub Actions
  co-authored change.

The repository does not automate stars, interact with unrelated maintainers, or
impersonate other contributors.
